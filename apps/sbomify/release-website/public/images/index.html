Container Images

postgres

Overview

A compliance-ready PostgreSQL container image built with Nix. This image is based on upstream nixpkgs PostgreSQL with additional CVE patches and license compliance configuration applied via custom Nix packages.

Pull command

docker pull ghcr.io/wellmaintained/postgres:latest

Base packages

  • PostgreSQL (from common/pkgs/ with upstream patches)
  • Runtime dependencies managed via Nix
